# DS_thesis_2025


## Code
All code is written in Python. The majority of the code is written and executed in Jupyter notebook, with some functions stored in .py files.

The following notebooks contain the code for the project:

- metro_network.ipynb: Extraction of metro features using OSMnx
- passengers.ipynb: Loading and processing of passenger data from Metroselskabet (not shared)
- OTM.ipynb: Loading and processing of data from Ørestadstrafikmodellen (not shared)
- areacharacteristics.ipynb: Loading and processing of neighborhood attributes. NOTE: A part of the notebook code use either big datasets or processed OTM data, which is not made available. These code cells will therefore not be able to run, unless the big datasets are loaded using the provided link in the written report (not applicable for processed OTM data).
- clustering_ensemble.ipynb: Clustering process and analysis.
- flow_network.ipynb: Construction of metro network, network analysis and OD flow analysis
